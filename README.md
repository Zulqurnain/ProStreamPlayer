# ProStreamPlayer

Android Radio/Media streaming library using a stable AAC decoder with RadioPlayerService for reliable audio playback.

## Features

- Stable AAC decoder integration
- RadioPlayerService for background playback
- Easy-to-use API for stream URLs
- Supports radio and media streams
- Background service with notification controls
- Min SDK: 21 | Target SDK: 35

## Usage

```kotlin
ProStreamPlayer.init(context)
    .setStreamUrl("https://your-stream-url.com/stream")
    .play()
```

## Getting Started

```bash
git clone https://github.com/Zulqurnain/ProStreamPlayer.git
cd ProStreamPlayer
# Open in Android Studio
```
